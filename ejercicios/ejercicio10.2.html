La etiqueta video; para especificar una ubicaion del archivo que se reproducira, puede usar el elemento <source> dentro del elemento <video>. (Esto debe reemplazar el atributo src en la etiqueta <video>de apertura.) Tambien puede utilizar varios elementos <source> para especificar que el video esta disponible en diferentes formatos.(debido a un error en el iPad, debe proporcionar el video mp4 c omo el rpimer formato. de lo contrario, puede que no inicie.)

src:este atributo especifica la ruta de acceso al video.

type:se debe utilizar este atributo para decirle al navegador que formato es el video. De lo contrario, descargara parte del video para ver si puede reproducir el archivo (que tomara tiempo y ancho de banda).

codecs:el codec que se uso para codificar el video se suministra dentro del atributo type. Tenga en cuenta el uso de comillas simples, asi como de comillas dobles en el atributo de tipo, cuando se suministra.
