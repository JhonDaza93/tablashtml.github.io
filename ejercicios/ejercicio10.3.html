<audio> HTML5 introdujo el elemento <audio> para incluir archivos de audio en sus paginas. Al igual que con el video HTML5, los navegadores esperan diferentes formatos para el audio. El elemento <audio> tiene un numero de atributos que le permiten controlar la reproduccion del audio.

src:Este atributo especifica la ruta de acceso archivo del audio.

controls:Este atributo indica si el reproductor debe mostrar los controles.Si no utiliza este atributo, no se mostraran xontroles de forma predeterminada. Tambien puede especifivar sus propios controles mediante JavaScript.

Autoplay:La presencia de este atributo indica que el audio debe empezar a reproducirse automaticametne. (se considera una mejor practica permitir que los visitantes eligan reproducir el audio.)

Loop:este atributo especifica que la pista de audio debe reproducirse de nuevo una vez que haya finalizado.
